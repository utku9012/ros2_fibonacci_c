# ros2_fibonacci_c
ROS 2 Fibonacci Action Server
Bu proje, ROS 2 kullanılarak oluşturulmuş, Fibonacci dizisi hesaplayan bir Action Server ve Client uygulamasıdır. Kullanıcıdan bir order (sıra) alır ve adım adım geri bildirim (feedback) göndererek diziyi hesaplar.

fibonacci_action_server: Hesaplamayı yapan ve geri bildirim gönderen node.

fibonacci_action_client: Hedefi gönderen ve sonuçları ekrana basan node.
